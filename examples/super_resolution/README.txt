By running 'create_model.py',
We create a SR model based on an ECCV'18 paper, "Fast, Accurate, and Lightweight Super-Resolution with Cascading Residual Network".
The following is the link to its implementation in Tensorflow:
https://github.com/barrelo89/CARN

To save the super resolution result, stb image library is used.
It is a pure C library, which requires no installation.
The following is the link to it:
https://github.com/nothings/stb
